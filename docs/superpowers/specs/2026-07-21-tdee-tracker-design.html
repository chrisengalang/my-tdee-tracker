TDEE Calculation Logic

Initial TDEE — Mifflin-St Jeor + Activity Multiplier

// BMR (kcal/day)
if male: BMR = 10 × weight_kg + 6.25 × height_cm − 5 × age + 5
if female: BMR = 10 × weight_kg + 6.25 × height_cm − 5 × age − 161

// Activity multiplier
sedentary=1.2 light=1.375 moderate=1.55 active=1.725 very_active=1.9

initial_tdee = BMR × multiplier

Adaptive TDEE — Rolling Window Recalculation

// Requires ≥14 days of logs in the window (default 28 days)
avg_calories = mean(calories_kcal) // over window
avg_weight_change_kg = (last_weight − first_weight) / window_weeks

// 1 kg body mass ≈ 7700 kcal surplus/deficit
weekly_surplus_deficit = avg_weight_change_kg × 7700
daily_surplus_deficit = weekly_surplus_deficit / 7

adaptive_tdee = avg_calories − daily_surplus_deficit

Example: avg 1,800 kcal/day, lost 0.5 kg/week → deficit = 550 kcal/day → adaptive TDEE = 1,800 + 550 = 2,350 kcal
